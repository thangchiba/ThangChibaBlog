## ご紹介 (Goshōkai)

Jingi Betaバージョン0.2.0のリリースを発表できることを嬉しく思います！このアップデートでは、ゲームのインターフェースとインベントリシステムに大幅な改良が加えられ、ゲームプレイ体験全体を向上させるための重要な要素となっています。

<YouTubeEmbed url="https://www.youtube.com/watch?v=mkFDlAXRNb4&t=48s" />

**新機能 (Shin kinō)**

- **Inventory System**: 新しいインベントリシステムでは、アイテムのドラッグ、装備、使用が可能になりました。アイテムの属性や使用方法を理解するためのツールチップも用意されています。この機能は、インディーモバイルゲームのアセットと、独自の要件に合わせて私が作成したカスタムスクリプトを組み合わせて作成しました。
- **Graphical User Interface (GUI)**: GUI の機能強化には、新しい HealthBar と Loading シーンが含まれており、ゲームがより直感的で視覚的に魅力的になりました。
- **Pet Mechanics**: 現時点では、ペットはゲーム世界で飼い主の後を追います。ペットの攻撃スクリプトはまだ実装していませんが、ロードマップには含まれています！

**開発ノート (Kaihatsu nōto)**

Jingiのネットワーキングの側面に取り組む準備を進めている中で、スケーラビリティとパフォーマンスを確保するために、現在のコードの約70%を書き直すことを想定しています。つまり、一部の機能はまだ開発中であり、今後のアップデートで洗練される予定です。

**フィードバック募集 (Fīdobakku boshū)**

皆様からのフィードバックは非常に貴重です！リリースされた機能に関するご質問や、今後のアップデートに関するご提案がありましたら、以下のコメント欄にお寄せください。皆様のご質問にお答えし、貴重なご意見を参考にさせていただきます。
